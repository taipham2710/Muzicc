# MUZICC

## Purpose
This repository is built step by step to demonstrate the full lifecycle of a
cloud-native web application, from local development to production deployment
on AWS.

## Phases
- Phase 0: Repository discipline & security
- Phase 1: Backend development (local)
- Phase 2: Frontend development (local)
- Phase 3: Docker & local integration
- Phase 4: CI pipeline
- Phase 5: Cloud deployment (AWS + EKS)
